## 💬 Usage

### 1.認証機能について
掲示板の利用には新規登録が必要です。名前、メール、パスワードを設定しましょう。

### 2.掲示板について
対応するカテゴリーを選択して、スレッドの投稿ができます。スレッドでは、レスを書き込むことができます。また、スレッド一覧画面でスレッド、及びレスの検索ができます。
